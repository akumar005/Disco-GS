# Disco-GS
This repository contains the dataset introduced in the paper “Disco-GS: Gaussian Splatting in Dynamic Color Lighting”, accepted at CVPR 2026.

## Dataset Description 
1. The dataset contains 25 videos of static scenes captured under various dynamic colored lighting patterns.
2. To support both quantitative and qualitative evaluation, we also provide recordings of the same scenes captured under white lighting, which we refer to as pseudo ground truth (Pseudo-GT).
3. The Pseudo-GT frames are included either at the beginning or at the end of each video sequence.
